공공데이터포털 기타 농업 API 매뉴얼

출처: 농촌진흥청, 농림축산식품부 (공공데이터포털)

1. 농기계 임대사업소 조회 (Farm Machinery Office)

전국 농기계 임대사업소의 위치 및 보유 장비 현황을 조회합니다.

  • URL: /farmMachinOffice
  • 파라미터:
    • siDo (필수): 시도명 (예: 경기도)
    • siGunGu (필수): 시군구명 (예: 수원시)

2. 원예특용작물 기술정보 (Horticulture Tech)

원예 및 특용작물의 재배 기술과 품종 정보를 제공합니다.

  • URL: /Nihhs_TechInfo
  • 파라미터:
    • serviceKey: 인증키
    • cntntsNo: 콘텐츠 번호 (상세조회 시)

3. 농업기상 주산지 기상분석 (Agri Weather Analysis)

주요 작물 주산지의 기상 상황을 분석하여 제공합니다.

  • URL: /frcPlpd
  • 파라미터: obsr_Spot_Code (관측지점코드)
