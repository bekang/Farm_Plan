가락시장 실시간 경매 API 매뉴얼

출처: 서울시농수산식품공사 (가락시장)

1. 개요

가락시장의 전자 경매 낙찰 정보를 실시간으로 조회할 수 있는 API입니다.

2. 엔드포인트

http://www.garak.co.kr/homepage/publicdata/dataJsonOpen.do

3. 요청 파라미터

파라미터 필수 값 / 설명
id 예 신청 승인된 ID (예: 5775)
passwd 예 비밀번호 (특수문자 주의)
dataid 예 data36 (경매낙찰정보 고정값)
s_date 예 조회 날짜 (YYYYMMDD)

4. 주요 응답 필드

  • PUM_NM: 품목명 (예: 양파, 배추)
  • U_NAME: 단위 (예: 20kg, 10kg 상자)
  • AV_P: 평균가 (원)
  • MAX_P / MIN_P: 최고가 / 최저가
⚠️ 트러블슈팅 가이드:
1. 응답이 비어있음: s_date가 일요일이거나 공휴일(휴장일)인지 확인하세요.
2. 비밀번호 오류: 구형 시스템이므로 비밀번호에 특수문자(!@#)가 있으면 URL 인코딩 문제로 로그인이 실패할 수 있습니다. 비밀번호 변경을 권장합니다.
