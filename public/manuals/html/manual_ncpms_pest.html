NCPMS 병해충 API 매뉴얼

출처: 농촌진흥청 NCPMS

1. 개요

병해충 예측 정보 및 도감 정보를 제공합니다.

2. 병해충 검색 (SicknsInfo)

  • URL: http://ncpms.rda.go.kr/npmsapi/sicknsInfoList
  • 파라미터:
    • apiKey (필수): 인증키
    • sicknsNm: 병해충명 (한글 URL 인코딩)
    • cropNm: 작물명

3. 병해충 예측 (Forecasting)

  • URL: http://ncpms.rda.go.kr/npmsapi/dbyhsForeList
  • 파라미터:
    • apiKey (필수)
    • sidoCode: 시도 코드 (법정동)
    • sigunguCode: 시군구 코드

4. 주의사항

API 응답이 느릴 수 있으며, serviceType=JSON 지원 여부를 반드시 확인하세요 (대부분 XML 기본).
