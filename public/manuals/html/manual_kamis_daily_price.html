KAMIS 일일 도소매 가격정보 매뉴얼

서비스명: 일일 도소매 가격 정보 (Daily Price)
출처: aT 한국농수산식품유통공사
검증 완료

1. 개요

주요 농축수산물의 일별 도매 및 소매 가격 정보를 제공합니다. (Open API)

2. API 엔드포인트

  • XML 방식: http://www.kamis.or.kr/service/price/xml.do
  • JSON 방식: http://www.kamis.or.kr/service/price/json.do

3. 요청 파라미터 (필수)

파라미터필수 여부설명비고
p_cert_keyYes인증키 (API Key)가입 시 발급
p_cert_idYes요청자 ID예: 7036 등
p_returntypeNo응답 포맷json 또는 xml
p_regdayYes조회 날짜YYYY-MM-DD 형식
p_item_category_codeNo부류 코드100(식량), 200(채소) 등
⚠️ 개발 시 주의사항:
  • 콤마(,) 제거 필수: 가격 데이터가 "1,500" 처럼 콤마가 포함된 문자열로 옴. 숫자 계산 전 반드시 콤마를 제거(Replace)해야 함.
  • 단일 일자 조회: 하루치 데이터만 조회 가능. 기간 조회 시 날짜를 반복(Loop)해서 호출해야 함.
  • 휴일 데이터 없음: 주말 및 공휴일에는 데이터가 없거나 001 코드가 반환됨.
