농사로 품종 정보 매뉴얼

출처: 농촌진흥청 (농사로)

1. 개요

국내 육성 품종의 상세 특성, 육성 년도, 재배 유의사항 등의 정보를 제공합니다.

2. API 엔드포인트

  • 목록 조회: http://api.nongsaro.go.kr/service/varietyInfo/varietyList
  • 상세 조회: http://api.nongsaro.go.kr/service/varietyInfo/varietyInfo

3. 주요 파라미터

파라미터 필수 설명
apiKey 예 발급받은 인증키
sVarietyNm 아니오 품종명 검색 (URL 인코딩 필수)
serviceType 아니오 응답 형식 (기본값: XML, 옵션: JSON)

4. 호출 프로세스

  1. 목록 조회 (varietyList) 호출하여 원하는 품종을 찾습니다.
  2. 결과 목록에서 cntntsNo (컨텐츠 번호)를 확인합니다.
  3. 상세 조회 (varietyInfo) 호출 시 cntntsNo를 파라미터로 넘겨 상세 정보를 받습니다.
