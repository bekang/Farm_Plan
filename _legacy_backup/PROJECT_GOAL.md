# 프로젝트 목표: 꿈을 그리는 농장

## 비전 (Vision)
공공데이터를 활용하여 농부가 농지, 작물, 자원을 효율적으로 관리할 수 있도록 돕는 사용자 친화적인 웹 기반 농장 관리 애플리케이션입니다.

## 핵심 전략 (Key Strategies)
1.  **플랫폼**: 웹 애플리케이션 (서버리스 / 정적 웹).
2.  **인프라**:
    -   **초기 (2~3일)**: GitHub Pages 호스팅 및 로컬 스토리지 사용.
    -   **확정 (3일 후~)**: **NAS 기반 자체 호스팅 및 DB 구축**. 모든 데이터와 서비스는 NAS를 통해 운영됨.
3.  **데이터 기반**: 주요 농업 공공 API 연동 (공공데이터포털, 농사로, NCPMS).

## 핵심 API 연동 (Core API Integration)
-   **공공데이터포털**: 기상 정보, 토양 환경 정보, 농산물 도매 시세.
-   **농사로**: 농작업 일정, 농업 기술 정보.
-   **NCPMS (국가농작물병해충관리시스템)**: 병해충 발생 예측 정보.

## 현재 상태 (Status)
-   **단계**: 기획 및 설계 (Planning & Design)
-   **현재 작업**: Q&A 서비스 계획 및 아키텍처 수립.
