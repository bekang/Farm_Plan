## Executive Summary

### 프로젝트 명칭
**농업 수익 최적화 플래너 (Agricultural Profit Optimizer)**

### 비전
한국 농업인들이 **데이터 기반 의사결정**을 통해 수익성을 극대화하고, 불확실성을 최소화할 수 있도록 지원하는 스마트 영농 도구

### 핵심 가치 제안
- 📊 **실시간 시장 데이터**: 가락시장 등 공공 API를 통한 실시간 가격 정보
- 🌤️ **기상 예측 통합**: 날씨 데이터 기반 재배 일정 최적화
- 💰 **수익 시뮬레이션**: 작물별 파종 시기에 따른 수익성 예측
- 📅 **통합 영농 캘린더**: 다중 필지 관리 및 일정 시각화
- 🗂️ **로컬 우선 (Local-First)**: 인터넷 연결 없이도 기본 기능 사용 가능

---

## 문제 정의

### 현재 농업인들이 겪는 문제점

#### 1. 정보의 단편화
- 가격 정보는 KAMIS, 날씨는 기상청, 재배 기술은 농촌진흥청 등 여러 출처에 분산
- 통합된 의사결정 도구의 부재

#### 2. 불확실한 수익성 예측
- 파종 시기 결정 시 수확기 가격을 예측할 수단이 없음
- 과거 데이터 분석 도구의 부족

#### 3. 복잡한 필지 관리
- 여러 필지에 다양한 작물을 재배할 때 일정 관리의 어려움
- 수작업 기록으로 인한 비효율성

#### 4. 비용 관리의 어려움
- 투입 비용(종자, 비료, 인건비) 추적의 어려움
- 실제 수익률 계산의 복잡성

## 타겟 사용자

### 주요 페르소나

#### 1. 중소규모 시설농가 (Primary)
- **특징**: 2-5개 필지, 하우스 재배, 연매출 5천만~2억
- **니즈**: 정확한 수익 예측, 작업 일정 최적화
- **디지털 친숙도**: 중간 (스마트폰 사용 가능)

#### 2. 다품목 재배 농가 (Secondary)
- **특징**: 여러 작물 동시 재배, 복잡한 일정 관리
- **니즈**: 통합 캘린더, 작물별 수익성 비교
- **디지털 친숙도**: 중상

#### 3. 귀농·귀촌 초보 농업인 (Tertiary)
- **특징**: 재배 경험 부족, 데이터 기반 학습 선호
- **니즈**: 표준 재배 정보, 가이드라인
- **디지털 친숙도**: 높음

### Backend
```python
Framework: Python 3.10+
UI Framework: Streamlit 1.30+

API Clients: Requests, Python-dotenv
Validation: Pydantic 2.5+
Logging: Loguru

```
